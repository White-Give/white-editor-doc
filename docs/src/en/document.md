# Document

## Document Configuration

```js
document: {
  title: 'Untitled Document',
  content: 'White-Editor is a modern WYSIWYG editor based on Vue 3 and tiptap.',
}
```

## Document Configuration Description

| Option | Type | Default | Description |
| --- | --- | --- | --- |
| title | string | Untitled Document | Title of the document container |
| content | string | White-Editor is a modern WYSIWYG editor based on Vue 3 and tiptap. | Content of the document container |
