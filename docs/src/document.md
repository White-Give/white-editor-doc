# document

## document 配置项

```js
document: {
  title: '未命名文档',
  content: 'White-Editor 是一个基于 Vue 3 和 tiptap 的现代所见即所得编辑器。',
}
```

## document 配置项说明

| 配置项 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| title | string | 未命名文档 | 文档容器的标题 |
| content | string | White-Editor 是一个基于 Vue 3 和 tiptap 的现代所见即所得编辑器。 | 文档容器的内容 |
